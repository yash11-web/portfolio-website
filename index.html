AskMyFile

AskMyFile is an advanced file-based Retrieval-Augmented Generation (RAG) chatbot built with Streamlit and powered by Google Gemini API. Users can upload a wide range of documents, including PDFs, Word documents, CSVs, Excel sheets, PowerPoint presentations, JSON files, and plain text files. The app automatically extracts the content from the uploaded files, splits it into meaningful chunks, and creates a FAISS vector store using Hugging Face embeddings to enable efficient semantic search.

Once the vector store is created, users can ask natural language questions about the content of their documents. The chatbot leverages the Gemini LLM to provide accurate and context-aware answers by retrieving relevant sections from the uploaded files. AskMyFile is designed for ease of use, allowing users to preview the extracted content, ask queries interactively, and view the source content used for generating answers.

Tech Stack: Python, Streamlit, LangChain, FAISS, Hugging Face, Google Gemini API
