AskMyFile

This is a file-based Retrieval-Augmented Generation (RAG) chatbot built with Streamlit. It allows users to upload documents (PDF, DOCX, CSV, etc.), extracts the content, creates a FAISS vector store using Hugging Face embeddings, and answers queries using a locally hosted GGUF language model(LLM).

Tech Stack: Python, LangChain, FAISS, Hugging Face, Streamlit

Data Visualization Assistant

Visualizer AI is an intelligent data visualization tool that allows users to generate insightful plots using natural language commands. Built with Streamlit, it uses a locally hosted LLaMA model to understand user queries and convert them into Python code using Matplotlib, Seaborn for interactive plots. This tool bridges the gap between non-technical users and complex data visualizations without writing code.

Tech Stack: Python, Matplotlib, Seaborn, Streamlit
